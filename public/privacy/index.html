SJV Marine Reporting System App

Privacy Policy

Last Updated: April 2026

1. Introduction

SJV Marine Reporting System ("we", "our", "us") is designed for maritime inspection reporting, survey checklist management, visiting card data extraction, and PDF/Excel report generation. This Privacy Policy explains how we collect, use, and protect your information when you use the application.

2. Data Collection and Usage

2.1 Account & Authentication Data

  • Email address and password for secure login
  • Device identifiers to manage sessions and enforce single-device login
  • Engineer name and basic user details

2.2 Device & Location Information

  • Device model and basic device information for app functionality
  • Temporary or approximate location during login or inspection
  • Location is used only for operational logging and not continuously tracked

2.3 Inspection & Survey Data

  • Vessel details, classification, and identifiers
  • Inspection timestamps and selected locations
  • Survey checklist responses and equipment details

2.4 Camera, Media & OCR Processing

  • Capturing inspection images using device camera
  • Capturing digital signatures and official stamps
  • Scanning visiting cards using OCR (Optical Character Recognition)
  • Extracting text data (name, phone, email, company) from scanned cards
  • Images are used only for report generation and data extraction

2.5 File Generation & Storage

  • Generation of PDF inspection reports
  • Generation of Excel files from scanned visiting card data
  • Files stored locally on device for preview and sharing

2.6 Network Communication

  • Internet access used for authentication and syncing reports
  • Secure transmission of inspection and report data to backend servers

3. How We Use Information

  • To generate maritime inspection and compliance reports
  • To extract structured data from visiting cards and export to Excel
  • To maintain accurate operational records
  • To improve system performance and reliability

4. App Permissions

4.1 Camera

Used for capturing inspection images and scanning visiting cards.

4.2 Storage / File Access

Used to save PDF reports, Excel files, and selected media.

4.3 Location

Used for recording inspection-related location inputs.

4.4 Internet Access

Required for authentication, syncing, and report submission.

5. Data Storage & Retention

  • Temporary data stored locally on the device
  • Inspection data transmitted securely to backend systems
  • Generated reports may be retained for operational use
  • Temporary files are periodically cleared

6. Data Sharing

We do not sell, rent, or trade your personal or inspection data.

  • Data is shared only with authorized backend systems for processing
  • Data may be disclosed if required by law or regulatory authorities

7. Data Security

  • All communications are secured using HTTPS encryption
  • Access to files is restricted using platform-level security
  • Measures are in place to prevent unauthorized access

8. User Rights

  • You may request access to your data
  • You may request correction or deletion
  • You may stop using the application at any time

9. Third-Party Services

  • OCR processing libraries for text extraction
  • Networking libraries (e.g., Retrofit / OkHttp)
  • Android system APIs (MediaStore, File handling)

10. Children's Privacy

This application is not intended for individuals under 18 years of age.
